Práctica 1 – Introducción a PHP + Ejercicios Extra

Este repositorio contiene todos los ejercicios de la Práctica 1 de Introducción a PHP, incluyendo los ejercicios extra. Cada ejercicio se encuentra en un archivo independiente (Exercise01.php, Exercise02.php, etc.), siguiendo las indicaciones del enunciado.

Contenido de los ejercicios
Archivo	Descripción
Exercise01.php	Declara dos variables numéricas, realiza operaciones básicas, compara las variables y calcula el área de un triángulo si ambas son mayores que 1.
Exercise02.php	Usa una variable numérica y un switch/case para mostrar el día de la semana correspondiente. Indica si el valor no es válido.
Exercise03.php	Trabaja con bucles (for, while, do/while) usando dos variables mayores de 10. Muestra números pares y cuenta hacia atrás según el enunciado.
Exercise04.php	Genera un número aleatorio entre 1 y 6 y calcula su raíz cuadrada.
Exercise05.php	Genera números aleatorios entre 0 y 20 hasta que la suma supere 100. Muestra la cantidad de números pares e impares.
Ejercicios Extra
Archivo	Descripción
Exercise06.php	Muestra todos los números entre 1 y 50 que sean pares y múltiplos de 3.
Exercise07.php	Muestra los números del 200 al 300 avanzando en potencias de 4, sumando 4 elevado a cada potencia sucesiva.
Exercise08.php	Muestra los primeros 20 números impares empezando desde 1.
Exercise09.php	Genera 20 notas aleatorias entre 0 y 10, clasifica las notas (suspenso, aprobado, notable, sobresaliente), suma todas las notas y calcula la media.
Exercise10.php	Calcula el impuesto de una herencia según los tramos del enunciado. Incluye ejemplos con 6.000 € y 75.000 € para validar el cálculo.
Requisitos

PHP 7.4 o superior.

Servidor local (XAMPP, WAMP, MAMP o similar) para ejecutar los scripts.

Uso

Clonar el repositorio:

git clone https://github.com/Ignaciobrenas/EjerciciosINTRO_PHP


Copiar los archivos al directorio del servidor local (htdocs o equivalente).

Acceder a cada archivo desde el navegador, por ejemplo:

http://localhost/Exercise01.php


Seguir las instrucciones de cada ejercicio y observar los resultados.
